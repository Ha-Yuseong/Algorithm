# 알고리즘 푼 목록

### 알고리즘 해석은 코드 맨 아래 주석으로 있습니다. ###

#### 디렉토리 설명 ####

| 디렉토리 | 설명 |
| -------- | ---------- |
| backjoon | 백준 문제 코드 및 해설 |
| before_algo | 예전에 풀었던 문제들의 코드 해설이 없습니다. |

backjoon : 백준 문제 코드 -> 맨 밑의 주석으로 풀이를 포함하고 있습니다.

before_algo : 예전에 풀었던 알고리즘 문제들의 코드 모음집을 현재 레포에 올려놓았고 해설이 달려있지 않습니다.
